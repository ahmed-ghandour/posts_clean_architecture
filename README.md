# posts_clean_architecture

 An Posts App Module which firstly fetches posts through an API and give you the ability to add,update and delete posts following the clean architecure in wrirting the codebase,I have used flutter_bloc,http,shared_preferences,dartz,get_it,equatable and internet_connetion_checker.
